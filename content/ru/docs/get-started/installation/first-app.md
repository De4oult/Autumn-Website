title: Первое приложение
description: Создайте минимальное Autumn-приложение и запустите его локально.
---

Первое приложение может быть совсем маленьким. Важно то, что объект приложения становится единой точкой входа для маршрутов, конфигурации и будущих модулей.

## Минимальная конфигурация

```python
from autumn import Autumn

import uvicorn

app = Autumn()

if __name__ == '__main__':
    uvicorn.run(app)
```

## Почему стоит начать именно так

- `Autumn(...)` задаёт чёткую границу приложения.
- Runtime остаётся явным, без скрытой CLI-магии.
- Когда позже появятся контроллеры и сервисы, у них уже будет стабильный корень.
