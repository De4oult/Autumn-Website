---
title: REST-контроллеры
description: Контроллеры определяют HTTP entry points и должны оставаться сфокусированными на транспортном поведении.
---

Используйте контроллеры там, где нужна чёткая HTTP-граница. Их задача — принять запрос, вызвать слой приложения и оформить ответ.

## Типовая форма

```python
@REST(prefix = '/users')
class UsersController:
    @get('/')
    async def list_users(self) -> JSONResponse:
        ...
```

## Как сохранять контроллеры здоровыми

- Держите handlers тонкими.
- Выносите ветвящуюся бизнес-логику в сервисы.
- Возвращайте явные response-объекты, когда транспортный контракт важен.
